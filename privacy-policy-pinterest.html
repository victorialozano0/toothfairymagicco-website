Privacy Policy — TFMC Pin Automation

Last updated: March 25, 2026

1. Introduction

TFMC Pin Automation ("the App") is a personal-use application developed by Tooth Fairy Magic Co. to manage Pinterest pins for our own Etsy shop. This privacy policy explains how the App handles data accessed through the Pinterest API.

2. Data Collection and Use

The App accesses the following data exclusively through the Pinterest API:

  • Pinterest Board Information: Board names and IDs to publish pins to the correct boards.
  • Pin Data: Pin creation and basic analytics for pins created by the App.
  • User Account Information: Basic account details for authentication purposes only.

This data is used solely to create and manage pins for the Tooth Fairy Magic Co. Etsy shop. No data from other Pinterest users is collected, stored, or processed.

3. Data Storage

The App runs locally on our own machine. Pinterest API data is processed in-memory and is not stored in any external database or cloud service. Authentication tokens are stored securely in local environment variables.

4. Data Sharing

We do not sell, trade, rent, or share any data accessed through the Pinterest API with any third parties. The data is used exclusively for managing our own Pinterest content.

5. Third-Party Services

The App interacts with the following services:

  • Pinterest API (v5): To create and manage pins on our own boards.
  • Etsy: Our e-commerce platform where products are listed (links in pins point to our Etsy shop).

6. User Rights

This App is for personal/organizational use only and does not have external users. The App owner can revoke Pinterest API access at any time by disconnecting the application from their Pinterest account settings.

7. Security

We implement reasonable security measures to protect data accessed through the Pinterest API, including secure storage of API credentials and running the application in a controlled local environment.

8. Children's Privacy

The App does not collect any personal information from children or any individuals. It is a backend tool for managing our own business content on Pinterest.

9. Changes to This Policy

We may update this privacy policy from time to time. Any changes will be reflected on this page with an updated date.
